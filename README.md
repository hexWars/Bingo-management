## 简介

BINGO 全国高校被试招募平台是一款连接测试供给方与需求方的科研平台，
具有耦合数据资源、简化招募程序的优势，并提供量表与数据库供用户参考，最
大化满足主试与被试双方的需求，致力于为用户提供更真实、更高效、更便捷、
更准确的调研数据与科研服务。
